# MongoCabra

## Instalação
``` sh
npm install -g mongo-cabra
```


## Modo de Uso
``` sh
mongo-cabra [options]
```

## Opções
```
-h, --help               output usage information
-V, --version            output the version number
-P, --port [dbport]      Informa a porta do banco de dados
-h, --host [dbdomain]    Informa o host do banco de dados
-u, --username [dbuser]  Informa o usuario do banco de dados
-p, --password [dbpass]  Informa a senha do banco de dados
-d, --database [db]      Informa o nome do banco de dados
-o, --output <file>      Exporta os dados para um arquivo
-i, --input <file>       Importa os dados de um arquivo
-c, --collection [name]  Seleciona o nome da collection
```
